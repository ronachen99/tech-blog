## Technology

The application utilizes a range of technogies to delivere its functionality. These include:

- Back-End Technologies:

  - Node.js
  - Express

- Front-End Technology:

  - Bulma

- Templating Engine:

  - Handlebars

- Database:

  - MySQL

- Object-Relational Mapping

  - Sequelize

- Environmental Variables:

  - Dotenv

- Encryption:

  - Bcrypt

- Deployment:

  - Heroku
